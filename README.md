# Air Sensor
这是一个使用 `Jetpack Compose` 开发的 Android 应用程序，它支持从搭载了[这个项目](https://github.com/camelbuba/airsensor_arduino)
的低功耗蓝牙设备中源源不断读取并显示温湿度数据。
